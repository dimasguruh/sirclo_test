SIRCLO project tes Guruh Sarwono

project ini dibuat dengan bahasa pemrograman PHP, dan database mysql

untuk membuka program dibutuhkan webserver seperti xampp,wampp,mampp,etc

disini saya menggunakan xampp, dan mohon disesuaikan apabila user 
menggunakan yang lain,

langkah-langkah nya adalah :

1. import file database "project_tes.sql" ke database di komputer anda
   ingat ini menggunakan mysql
2. copy folder sirclo kedalam folder tempat webserver dapat meng-eksekusi project
   disini sy menggunakan xampp, maka folder dipindahkan kedalam folder xampp/htdocs
3. setelah selesai langsung akses project saya dengan memasukan url://localhost/sirclo
   disana anda langsung diarahkan kepada tugas2 yang telah saya kerjakan
